Favicon Generator

How to generate favicon files:

  1. Download this SVG file: favicon.svg
  2. Go to RealFaviconGenerator
  3. Upload the SVG file
  4. Download the generated package
  5. Extract and copy all files to your public directory

After generating the files, you should have these in your public directory:

  • favicon.ico
  • favicon-16x16.png
  • favicon-32x32.png
  • apple-touch-icon.png
  • mstile-150x150.png
  • browserconfig.xml
